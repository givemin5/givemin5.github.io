單元 4

四種定位方法 :

  • Static - 預設值，無法使用 top / bottom / left / right 等屬性
  • Relative - 以自己的位置進行位移
  • Absolute - 會往外找到非 Static 的屬性以外的標籤進行定位，且不佔有空間(浮空)
  • Fixed - 不佔有空間，根據視窗位置定位
